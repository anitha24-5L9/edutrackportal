# 🎓 EduTrack Portal

EduTrack Portal is a Student Management System built using Flask and SQLite.

## Features

- Admin Login
- Student Login
- Add / Edit / Delete Students
- Attendance Management
- Attendance Analytics
- Branch-wise Analytics
- Top 5 Students Chart
- Student Dashboard
- Password Change Feature

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Chart.js
